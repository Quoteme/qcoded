Rocket

Just a little shoot 'em up. You need a joy pad for the 2 player mode. First insert the joy pad, then load the page, after that press a few buttons and at the end just start the game. You might need to repeat this a few times, as the game might not find your joy pad at first. (left controls = first player | right controls = sec. player)
Controls:
- Move = Arrow keys | directional buttons
- Shoot missile = 'S' key | 'A' button
- Place mine(after 1st level) = 'A' key | 'B' button
- Shoot laser(after 7th level) = 'Q' key | 'X' Button
- teleport (activate cheat) = 'W'/'D' key | not included yet
- triple shoot (after one players health is under 3) 'S' key | 'A' button

Jump 'n Runner

Jump and run in this game to the next level. More information: div class="spoiler">

Writing this game was actually my dream for ~2.5 years.
one of the very first games I made in summer 2013 (August 2013 i think) was called "Jump n runner". It included a character selection menu as well as 3 levels and never got out of the alpha version. The only "programming language" I knew was Visual Basic, so i wrote this platfromer witch very many gliches and bugs in VB. There were problems all over the place, like collision detection not working and input handeling (which had sometimes a second delay).
But I never stopped thinking about this game and so I starded this "reboot" of it ~2.5 years later with much more computer skills, knowledge and the awesome framework PHASER for javascript.
I hope you enjoy this game and if you have any ideas for levels (like a drawing of a potential level), I would be glad to include it as a stage in Jump 'n Runner.
